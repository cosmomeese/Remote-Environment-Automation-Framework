
customPeripheral: 	This folder contains the peripheral core code that is common to all peripherals. 
			Developers can use this code by filling in the "custom.ino" and "custom.h" template 
			files and adding their PeripheralGen files to this folder as discussed in the 
			Developer's Manual.


Demonstration System Peripherals: 	This folder contains the code used to program the peripherals 
					that will be used in our demonstration system.


blinkingPeripheralDemo:	This folder contains an example of a simple peripheral created using the code and 
			template files provided in customPeripheral.


To open any of the arduino projects open the arduino file (.ino) that has the same name as the folder it is in.
